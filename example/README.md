# Garnish Example

A small Flutter app demonstrating the [`garnish`](../) package.

Pick a base color and watch Garnish:

- generate readable text on top of it (**Auto Contrast**),
- report its luminance, classification, and contrast ratios (**Color Analysis**),
- flag WCAG AA/AAA compliance, and
- build a harmonious palette (**Generated Palette**).

## Run

```bash
flutter pub get
flutter create .   # first run only — generates the platform runners
flutter run
```
